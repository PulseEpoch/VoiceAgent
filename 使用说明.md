# 语音终端使用说明

## 快速开始

```bash
# 1. 直接运行（首次会自动下载模型，约 71MB）
python voice_agent.py

# 等待提示：[MLX-Whisper] 已连接 Whisper Server

# 2. 按住 Ctrl+K 说话，松开自动识别并执行
```

## 工作原理

```
你按住 Ctrl+K → 开始录音 → 松开 Ctrl+K → 
发送到 Whisper Server → 识别文字 → 自动输入到终端 → 执行
```

## ModelScope 模型说明

默认使用 **mlx-community/whisper-tiny-mlx**（国内下载快）

### 首次运行
- 自动从 ModelScope 下载模型（71MB）
- 下载时间：8-15 秒（国内网络）
- 缓存位置：`~/.cache/modelscope/hub/models/`
- 只需下载一次，之后直接使用缓存

### 可选模型

| 模型 | 大小 | 速度 | 准确度 | 适用场景 |
|------|------|------|--------|----------|
| whisper-tiny-mlx | 71MB | 最快 | 一般 | 日常快速命令 |
| whisper-base-mlx | 142MB | 快 | 良好 | 推荐日常使用 |
| whisper-small-mlx | 483MB | 中等 | 很好 | 高准确度需求 |

切换模型：
```bash
python voice_agent.py --model mlx-community/whisper-base-mlx
```

## 常见问题

### Q: 首次运行很慢？
A: 正在下载模型，只需等待一次。后续启动秒开。

### Q: 识别错误？
A: 尝试使用更大的模型：
```bash
python voice_agent.py --model mlx-community/whisper-base-mlx
```

### Q: 如何停止？
A: 正常退出终端即可（Ctrl+D 或 exit），程序会自动清理。

### Q: 多个终端可以共用吗？
A: 可以！手动启动服务器：
```bash
# 终端1
python whisper_server.py --model mlx-community/whisper-base-mlx

# 终端2、3、4...
python voice_agent.py --no-auto-start
```

## 性能优化

### 使用 Unix Socket（推荐）
延迟更低：
```bash
python voice_agent.py --socket /tmp/whisper.sock
```

### 选择合适的模型
- 快速响应：tiny
- 日常使用：base
- 高准确度：small
